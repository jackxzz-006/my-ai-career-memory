# 🚀 CareerTwin AI

> **Your AI-Powered Digital Career Twin** – Personalized career guidance, skill tracking, roadmap generation, and AI-driven learning assistance.

## 📌 Overview

CareerTwin AI is an intelligent career development platform that acts as a **digital twin** for students and professionals. It analyzes users' skills, interests, and career goals to generate personalized learning roadmaps, recommend projects and certifications, track progress, and provide AI-powered career insights.

The platform helps users bridge the gap between their current skills and their dream career by offering customized guidance, real-time progress tracking, and interactive AI assistance.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/jackxzz-006/CareerTwin-AI.git
```

Navigate into the project:

```bash
cd CareerTwin-AI
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

---

## 🔧 Environment Variables

Create a `.env` file in the project root and configure the required variables.

Example:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_key
VITE_OPENAI_API_KEY=your_api_key
```
